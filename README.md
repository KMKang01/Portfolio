# 강경민 Portfolio (Vue + Vite)

AiTStory 플랫폼 개발 직무 지원용 포트폴리오 웹사이트입니다.

## 실행 방법

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```

## Vercel 배포

1. GitHub 저장소 생성
2. 이 프로젝트 업로드
3. Vercel에서 Import Project
4. Framework Preset은 Vite로 자동 인식
5. Deploy

## 수정할 부분

- `src/App.vue` 하단 GitHub 링크
- `src/App.vue` 하단 Demo / Video 링크
- 필요 시 프로젝트 상세 내용 및 연락처 수정
